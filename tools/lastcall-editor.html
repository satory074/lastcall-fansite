ここに lastcall.json をドロップ 読み込み中…

データを自動で読み込んでいます…

自動読み込みできませんでした（file:// で直接開いた場合はブラウザの制限で自動読み込みできません）。

自動で読み込むには、ターミナルで npm run edit を実行してください（サーバが起動しブラウザが開きます。編集後は「ファイルに保存」で直接書き戻せます）。

このまま手動で使う場合は、上の「JSONを読み込み」から src/data/lastcall.json を選ぶ（またはドラッグ&ドロップ）→ 編集 → 「ダウンロード」で差し替え → npm run build → push。
